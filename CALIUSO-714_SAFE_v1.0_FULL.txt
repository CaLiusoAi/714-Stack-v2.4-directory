¬ß1. CORE STATE PRIMITIVE (CALIUSO-714)
------------------------------------------------------------

CALIUSO-714 is a conceptual way to describe reasoning as a state:

    S_t = (O_t, C_t, M_t)

    O_t > 0 : "Observables" = explicit structure
              (definitions, claims, examples, equations, code, tests).
    C_t >= 0 : "Hidden complexity" = what is still vague, assumed, or unresolved.
    M_t     : "Ledger" = an append-only log of important moves or updates.

We define a ratio:

    R_t = ln(C_t / O_t)

In practice, you do NOT need to literally compute R_t as a number.
Treat it qualitatively:

    - O_t UP when you make things clearer, more explicit, and better structured.
    - C_t UP when you add ambiguity, vague assumptions, or unexplained jumps.

CALIUSO-714's goal:

    "Over time, hidden complexity should NOT secretly grow
     relative to explicit structure."
